    • About MJ

    Michael Jackson known as the "King of Pop," Michael Joseph Jackson was a best-selling American singer, songwriter and dancer. As a child, Jackson became the lead singer of his family's popular Motown group, the Jackson 5. He went on to a solo career of astonishing worldwide success, delivering No. 1 hits from the albums Off the Wall, Thriller and Bad. In his later years, Jackson was dogged by allegations of child molestation. He died in 2009 at age 50 of a drug overdose just before launching a comeback tour.

    3 main defrencces between Web 1.0, Web 2.0 and Web 3.0:

    Diffrence Web 1.0 Web 2.0 Web 3.0
    Years 1990-2000 2000-2010 2010-2020
    Number of users Millions Billions Trillions
    User-web communication "readable" - Only limited interaction between sites and web users “writable” - Web 2.0 facilitates interaction between web users and sites allow users to interact more freely with each other “executable” - Dynamic applications, interactive services, and “machine-to-machine” interaction.
    The Main Features Hyper-linking and bookmarjing of the web pages. It only consisted of online guestbook and framesets. It allowed many users to publish their own content through few basic steps, which was not possible in the Web 1.0. 5 main features that can help us define Web 3.0: Semantic Web, Artificial Intelligence, 3D Graphics, Connectivity, Ubiquity.
    Examples Static websites: an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback. Youtube, Wiki, Flickr, Facebook (which encourages participation, collaboration, and information sharing). Tivo, a digital video recorder: its recording program can search the web and read what it finds to you based on your preferences.




    The difference by a use case of searching "side effects of drugs":

    Web 1.0 was essentially a repository of information that people could read passively, without being able to shape the information or add their own. There is no search engine. The user needs to know the exact URL and just if the website exists the user will get an information about the side effects of drugs.

    In Web 2.0, search engines such as Google and Yahoo! were prevalent for searching the Web. Search in dynamic pages, however, is ei- ther inexistent or far from perfect. Tagging was popularized by websites associated with Web 2.0 and is an important feature of many Web 2.0 services. Naive retrieval: only pages with the tags that connected to "medicine side effect" will be retrieved.

    In Web 3.0, thanks to the use of artificial intelligence, the potential for a blockchain-based solution for storing and sharing data, and the evolution of the semantic web to provide personalized experiences, the user can use the search engine by a query and then gets ranked pages that are relevant to the side effects of drugs, as he wanted.
